# Map続

Map続 is an experimental 3d webgl map viewer based upon three.js. It supports rendering in non web mercator projections.

The following [demo](http://demo.geoloep.nl/map3/) of map続 show two wms layers in the EPSG:28992 projection. Use the left and right mouse buttons to control the camera.
